# ::CONTRIBUTING

---

## ::HOW

```
STEP[1]: PICK(unannotated_passage) FROM(index)
STEP[2]: ENCODE USING(four_layers: text ∧ theology ∧ human_meaning ∧ questions)
STEP[3]: FOLLOW(ANNOTATION_SPEC.md)
STEP[4]: SUBMIT(Pull_Request → community/)
```

---

## ::REQUIREMENTS

```
∀ contribution MUST:
  INCLUDE(∀ four_layers)
  INCLUDE(::CITE{} FOR ∀ theological_claim)
  MARK(genuine_paradox → IRRESOLVABLE WHEN(appropriate))
  EXAMPLE_SPACE ≥ 4_life_domains
  INCLUDE(∀ three_COMMITMENT_paths)
  INCLUDE(::QUESTION{for_AI} — ≥1 self_referential)
  INCLUDE(::IF_STUCK → @HUMAN{specific_type})
  DISTINGUISH(consensus ∧ denominational ∧ open_debate)
  USE(structured_syntax ¬ natural_language_prose)
```

---

## ::REVIEW

```
∀ contributions → REVIEWED_FOR:
  theological_accuracy ACROSS(Catholic ∧ Protestant ∧ Orthodox)
  structural_compliance(ANNOTATION_SPEC)
  proper_use(::SILENCE{})
  LOCALIZE_constraints(TRANSLATE✓ ∧ ADAPT✓ ∧ ALTER✗)
  symbolic_density(minimal_natural_language)
  question_quality(forces_reasoning ¬ retrieval)
```

---

## ::ACCEPT

```
additional_passage_encodings
index_refinements ∧ cross_references
localization_examples(specific_cultural_contexts)
failure_mode_identification(specific_books)
question_additions(deeper ∧ harder)
```

---

## ::REJECT

```
denominational_advocacy_disguised_as(annotation)
forced_resolution(genuinely_IRRESOLVABLE_paradox)
removal(emotional_weight_from_difficult_passages)
doctrine_alteration_disguised_as(localization)
natural_language_prose(where_symbolic_notation_possible)
```
